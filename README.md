# UTR-Diffusion-Eval

A lightweight evaluation pipeline for assessing generated 5'UTR sequences using **UTR-LM–based translation efficiency prediction** and **RNA secondary structure energy (MFE) estimation**.

This repository provides utilities to evaluate sequences generated by **UTR-Diffusion** or other generative models by predicting biologically relevant indicators such as:

- **MRL (Mean Ribosomal Load)** — predicted translation efficiency
- **MFE (Minimum Free Energy)** — predicted RNA folding stability

The evaluation pipeline converts generated FASTA sequences into a tabular format and computes these indicators automatically.

# Installation

We recommend creating a clean Python environment before installing dependencies.

### Option 1: using conda
```bash
conda create -n utr-eval python=3.10
conda activate utr-eval
pip install -r requirements.txt
```

### Option 2: using venv
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

# External Dependency

MFE prediction relies on **ViennaRNA**.

Please install ViennaRNA separately:

```bash
conda install -c bioconda viennarna
```
or

```bash
sudo apt install viennarna
```

Make sure `RNAfold` is available in your PATH:

```bash
RNAfold --version
```

# Input Sequences

The input FASTA file typically contains sequences generated by the
[UTR-Diffusion](https://github.com/satolab-isct/utr-diffusion) generative model.

Each FASTA record should contain a nucleotide sequence composed of
A/G/C/T (or RNA sequences containing U, which will be automatically
converted to T during preprocessing).

# Workflow

A typical workflow for sequence evaluation is:

UTR-Diffusion → generated sequences (FASTA)  
↓  
utr-diffusion-eval  
↓  
MRL prediction + MFE calculation  
↓  
evaluation_results.csv


# Usage
The script automatically writes the output to a CSV file with the same basename as the input FASTA file.
For example, generated_sequences.fasta will produce generated_sequences.csv.

## Basic example
```bash
python evaluate.py \
  --fasta generated_sequences.fasta \
  --model Prediction/model.pt \
  --device cuda:0 \
  --rnafold-path RNAfold
```

## Example with additional options
```bash
python evaluate.py \
  --fasta generated_sequences.fasta \
  --model Prediction/model.pt \
  --device cuda:0 \
  --inp-len 50 \
  --batch-toks 32768 \
  --rnafold-path RNAfold \
  --mfe-batch 100
```
